## 🏠 부동산 채팅

{% hint style="info" %}
#### 부동산 내에서 <mark style="color:green;">**/rc**</mark> 또는 <mark style="color:green;">/res rc \[부동산]</mark> 으로 해당 부동산 채팅에 입장할 수 있습니다.
{% endhint %}

{% hint style="warning" %}
부동산 채팅에 입장을 하기 위해선 해당 부동산의 chat 권한이 필요합니다.
{% endhint %}

{% hint style="warning" %}
입장하려는 부동산과 같은 월드에 있어야 합니다.
{% endhint %}
