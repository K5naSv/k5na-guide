### 명령어

* `/jobs info 직업 행동` : 해당 직업 블록/행동의 소득과 경험치 보기
* `/jobs join 직업` : 해당 직업 가입
* `/jobs leave 직업` : 해당 직업 탈퇴
* `/jobs stats 닉네임` : 해당 플레이어의 직업 레벨 보기
* `/jobs top 직업` : 해당 직업 순위 보기
