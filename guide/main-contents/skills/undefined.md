# 활동 종류 및 능력



{% tabs %}
{% tab title="🌾 농사" %}
#### **숙련도 획득 방법**

* 작물 수확
* 열매 수확
* 버섯 채집

#### **패시브 능력**

> 자동으로 적용되는 **패시브 버프이며,** \
> **레벨 2에 해금** 이후 **5레벨마다 강화**됩니다.

* 풍성한 수확 (작물 드롭량 증가)
* 농부 (추가 숙련도 획득)
* 유전학자 (식물성 음식 포만감)
* 성장의 오라 (농작물 성장)

#### **마나 능력**

> 사용 시 **마나를 소모**하고, \
> **레벨 6에 해금** 이후 **6레벨마다 강화**됩니다.

작물을 수확한 자리에 자동으로 씨앗이 심어집니다.

* 괭이를 들고 <mark style="color:blue;">**우클릭**</mark> 후 작물 수확으로 발동합니다.
* 능력 발동 후 <mark style="color:blue;">**5초**</mark>부터 레벨에 따라 최대 <mark style="color:blue;">**80초**</mark>까지 지속됩니다.
* 쿨타임은 레벨에 따라 상이하며 최소는 <mark style="color:blue;">**125초**</mark>입니다.
