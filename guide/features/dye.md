# 🧊 염색

{% hint style="info" %}
염색 기능을 이용하여 간편하게 아이템을 염색해 보세요!\
<mark style="color:blue;">**손에 들고 있는**</mark> 아이템 혹은 <mark style="color:blue;">**바라보는 블록**</mark>을 염색할 수 있습니다.
{% endhint %}

### 염색 명령어

<table><thead><tr><th width="221">명령어</th><th>설명</th></tr></thead><tbody><tr><td><strong>/염색 아이템 [색상] [숫자]</strong></td><td>해당 색상으로 손에 들고 있는 아이템을 <mark style="color:blue;"><strong>염색</strong></mark>합니다.<br>숫자는 <mark style="color:blue;"><strong>1~64</strong></mark>까지 입력할 수 있습니다.</td></tr><tr><td><strong>/염색 아이템 [나무] [숫자]</strong></td><td>해당 나무 종류로 손에 들고 있는 나무를 <mark style="color:blue;"><strong>변경</strong></mark>합니다.</td></tr><tr><td><strong>/염색 블록 [색상]</strong></td><td>해당 색상으로 <mark style="color:blue;"><strong>바라보고 있는</strong></mark> 블록의 색상을 <mark style="color:blue;"><strong>염색</strong></mark>합니다.</td></tr><tr><td><strong>/염색 블록 [나무]</strong></td><td>해당 나무 종류로 <mark style="color:blue;"><strong>바라보고 있는</strong></mark> 블록의 나무를 <mark style="color:blue;"><strong>변경</strong></mark>합니다.</td></tr></tbody></table>
