# 🔄 블록 회전

{% hint style="info" %}
계단, 원목, 유광 테라코타 등을 <img src="../../.gitbook/assets/shift_key.png" alt="" data-size="original">+<img src="../../.gitbook/assets/right_click.png" alt="" data-size="line">으로 회전할 수 있습니다.
{% endhint %}

{% hint style="warning" %}
부동산 내에서는 <mark style="color:blue;">**설치(place),**</mark> <mark style="color:blue;">**파괴(destroy)**</mark> 플래그 권한이 없으면 회전이 <mark style="color:red;">**불가**</mark>합니다.
{% endhint %}
