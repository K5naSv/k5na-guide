# 🛏️ 셋홈

{% hint style="info" %}
[등급](../main-contents/rank.md)에 따라 최대 셋홈 설정 가능 갯수가 다릅니다 (등급 가이드)
{% endhint %}

{% hint style="warning" %}
한글 이름은 가능하나, 특수문자를 홈이름으로 설정할 경우 오류가 발생될 수 있습니다.&#x20;
{% endhint %}
